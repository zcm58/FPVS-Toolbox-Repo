    Test Selection¶

    Pick the smallest relevant tests first, then broaden when the change affects shared behavior.

    Marker Shortcuts¶

    The repository auto-applies common pytest markers from test filenames during collection. Use these for quick local selection:

    python -m pytest -m "not slow" -q
python -m pytest -m gui -q
python -m pytest -m stats -q
python -m pytest -m project_io -q
python -m pytest -m processing -q
python -m pytest -m smoke -q

    Available markers are declared in pyproject.toml: gui, stats, project_io, processing, plot_generator, ratio, smoke, integration, slow, and qt.

    Markers are selection aids, not a substitute for the focused test lists below. When changing a specific module, run the nearest named test first.

    Removed Source Localization¶

    Source Localization/eLORETA has been removed from active runtime. Do not add tests that import Tools.SourceLocalization, restore availability shims, or assert GUI/settings behavior unless restoration is explicitly scoped as a new feature. Do not add tests that require bundled fsaverage MRI template data; that template is no longer retained in active source or quarantine.
