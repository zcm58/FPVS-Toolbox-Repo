# Run a Study

Use this page when you want the shortest route from project setup to usable outputs.

The pages in this section are practical workflow guides. They focus on what to do in the app and what to check before moving to the next step.

| Page | Use this page when... |
|---|---|
| [Project Setup](project-setup.md) | You need to create a project, choose folders, or enter trigger codes. |
| [Processing Pipeline](processing-pipeline.md) | You want to understand what happens after you start processing. |
| [Output Files](outputs-reference.md) | You need to find or interpret exported Excel files. |
| [Troubleshooting](troubleshooting.md) | Something is missing, blocked, or producing unexpected results. |

## Recommended order

1. Create the project and event map.
2. Process one participant as a pilot.
3. Confirm Excel outputs.
4. Process the full dataset.
5. Move to [Understand Results](../results/understand-results.md).
