# Individual-Level Detectability

The Individual-Level Detectability tool creates participant-level figures for reviewing FPVS responses.

## What It Does

Use this page to document the participant-level topomap and spectrum outputs generated by the tool.

## When To Use It

Use this tool after processing when individual participant response patterns need to be reviewed or reported.

## Basic Inputs

- Processed participant Excel outputs.
- Harmonic or frequency settings.
- Figure export settings.

## Basic Outputs

- Participant-level detectability figures.
