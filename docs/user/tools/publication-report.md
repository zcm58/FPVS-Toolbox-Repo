# Publication Report

The Publication Report tool generates a report bundle from processed FPVS Toolbox outputs.

## What It Does

Use this page to document the report bundle, included tables, and any diagnostics generated for manuscript review.

## When To Use It

Use this tool after processing and statistics exports are complete for a single-group project.

## Basic Inputs

- Processed FPVS Toolbox outputs.
- Statistics outputs or stats-ready workbooks, when required.
- Report settings selected in the tool.

## Basic Outputs

- Report documents or markdown.
- Data workbooks and audit files.
- Logs or warnings that describe unavailable optional outputs.
