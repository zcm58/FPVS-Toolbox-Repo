# Publication Maps / Scalp Maps

Publication Maps, shown in the app as Scalp Maps, exports condition-level scalp-map figures from processed FPVS Toolbox outputs.

## What It Does

Use this page to document the basic scalp-map export workflow and the metrics shown in the generated figures.

## When To Use It

Use this tool after processing has produced condition-level Excel workbooks and after the project settings needed for harmonic selection are available.

## Basic Inputs

- Processed condition folders from FPVS Toolbox.
- Project frequency settings.
- Metric and color-scale settings.

## Basic Outputs

- Scalp-map figure files.
- Source-data or audit workbooks documenting plotted values.
