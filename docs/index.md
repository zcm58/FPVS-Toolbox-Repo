# FPVS Toolbox Documentation

Use this site when you need practical help processing FPVS EEG data, finding outputs, or choosing analysis steps without writing custom scripts.

FPVS Toolbox is a desktop app for processing FPVS EEG data and running statistics without writing custom analysis scripts.

!!! tip "New to FPVS or EEG analysis?"
    Start with [Getting Started](getting-started.md), then read [FPVS Concepts for Non-Experts](fpvs-concepts.md).

## What this documentation site covers

- How to set up and run an FPVS project from raw `.bdf` files to final results.
- What each analysis feature does and when to use it.
- Where output files are saved and how to interpret them.
- Common errors and practical fixes.

## Quick start path (about 10-20 minutes)

1. Read [Getting Started](getting-started.md).
2. Follow [First End-to-End Walkthrough](tutorial.md).
3. Review [Feature Tour](feature-tour.md) to learn what to run next.
4. Use [Output Files](outputs-reference.md) to find your exports.
